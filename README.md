# claude-agente

Mi primer repositorio creado con Claude Code.

## Descripción

Este proyecto es un espacio de práctica para aprender a trabajar con Claude Code y GitHub desde la terminal.

## Requisitos

- [GitHub CLI](https://cli.github.com)
- Git

## Uso

Clona el repositorio y empieza a explorar:

```bash
git clone https://github.com/subterraneogroup2025-netizen/claude-agente.git
cd claude-agente
```
